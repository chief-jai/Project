1. Install Latest version of Node JS.
2. Open a command Prompt and navigate to this project's location.
3. Run the command 'npm install'
4. Once the npm installs the dependencies, Run the command 'node main.js'. This should start the server.
5. To make sure the API works, Open this URL in browser - http://localhost:8081/listUsers. It should return the file's content.
6. File name is hardcoded in main.js file. Keep the file name in sync b/w main.js and the file itself.
7. Once the server is up, Open the index.html file in browser.
